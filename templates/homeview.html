How does it work?

Create an planner

Have a planner for an individual. Enter a name of the planner. Typically, it is the kid's name. And optionally enter a description for the planner.

Upload Pictures

Create a set of pictures for the planner. Picture can be a person, things, etc. They help your kid to know the activities ahead of time.

Setup the schedule

Plan the schedule by adding pictures and setting up duration for each activities. Some tasks are repeatable tasks while others are not. You have the flexibility to plan the schedule accordingly.
